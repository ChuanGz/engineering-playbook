# Standards

These standards govern every accepted document. Quality and terminology review happen before editorial polish.

## Standards

- [Document contract](document-contract.md) — required context and quality gates before a document is created.
- [Writing standard](writing-standard.md) — language and readability requirements.
- [Content quality](content-quality-standard.md) — evidence and decision-value requirements.
- [Terminology](terminology-standard.md) — canonical and responsibility-based language.
- [Decision records](decision-record-standard.md) — lifecycle and evidence for consequential decisions.
- [Contribution review](contribution-review-standard.md) — acceptance and rejection controls for changes.
